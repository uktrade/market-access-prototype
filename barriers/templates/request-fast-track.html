How to request a fast track response

Call this 24-hour number: 0800 123 4567

We will only be able to process calls which can demonstrate clear evidence that there will be significant and imminent financial or other loss, with action needing to be taken within 48 hours.

If your problem does not meet these criteria, please complete the report a trade barrier form. A relevant expert will respond to you within five days.

Only make a fast track request if:

- you are facing significant and imminent financial or other loss
- you need action to be taken within the next 48 hours
